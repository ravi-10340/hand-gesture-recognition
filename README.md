# Hand Gesture Recognition

## 🚀 Project Overview
This project detects hand gestures in real-time using OpenCV and MediaPipe.

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ How to Run
pip install -r requirements.txt
python main.py

## 🎯 Features
- Real-time hand tracking
- 21 landmark detection
- Webcam-based recognition
